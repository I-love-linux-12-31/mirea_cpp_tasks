**Поля в БД:**
 * ID : число
 * Имя : строка
 * Фамилия  : строка
 * Отчество  : строка
 * Курс  : число
 * Группа : строка
 * Оценки  : строка
 * Год начала учёбы  : число
 * День рождения  : строка

Оценки хранятся в виде строк из чисел 2345, F - незачёт и P - зачёт
#- Разделитель семестров
**Пример :** "54323455PPF#5555PP#2222FFF"
